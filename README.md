# PygameBackround

Animowane, subtelne tlo blockchain napisane w pygame.

## Uruchomienie

```powershell
pip install -r requirements.txt
python my_background.py
```

Sterowanie w oknie demo:

- `Space` - delikatna fala nowego bloku
- `F` - delikatny efekt fork
- `Tab` - zmiana trybu aktywnosci tla
- `Esc` albo `Q` - wyjscie
